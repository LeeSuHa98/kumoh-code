이프로그램은 야구게임을 콘솔 형태와 콘솔형태를 이용하여 GUI를 붙인것으로 2가지 프로그램이 들어있습니다.

-콘솔 실행  자바 파일 :BaseBall.java 이것만 컴파일해서 실행시키면 콘솔로 돌아간다.

-GUI로 구현된 자바파일 
  +BaseBallwithGUI.java  : 콘솔로 짠 자바파일과 거의 비슷하다고 보면 된다.GUI를 위해 약간 수정했음
  +Gui_Baseball.java : GUI를 구현..이파일이 메인 파일이다.. 거기에 각각의 이벤트를 걸고.. BaseballwithGUI.java 객체를 생성.  Dial_log.java  객체도 생성해서.. 필요시에 쓸수 있게 구현.
  +Dial_log.java : 자바 어플리케이션 실행시..여러가지 이벤트 발생때마다 다이얼 로그를 띄우기  위해 구현.
